# 打包内容说明

本 ZIP 包为脚本和文档交付包，不包含已打包 EXE、不包含 `build/`、`dist/`、不包含大型 demo CSV。

## 包含内容

代码：

- `ebeam_backend.py`
- `ui_app.py`
- `plot_cd_by_radius.py`
- `plot_layer_cd_distribution.py`
- `plot_line_cd_loading.py`
- `plot_wafer_heatmap.py`
- `generate_demo_data.py`
- `build_exe.ps1`
- `requirements.txt`

文档：

- `README.md`
- `脚本使用说明.md`
- `工程详细说明.md`
- `架构实现说明.md`
- `打包内容说明.md`

## 内网打包 EXE

```powershell
python -m pip install -r requirements.txt
.\build_exe.ps1
```

输出：

```text
dist\EbeamDataVisualizer\EbeamDataVisualizer.exe
```

需要复制整个 `dist\EbeamDataVisualizer` 文件夹。
