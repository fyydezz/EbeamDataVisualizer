# Ebeam Data Visualizer 使用说明

本文档面向日常使用者，说明如何安装、运行 UI、选择图表、使用命令行脚本，以及如何处理常见报错。

## 1. 安装依赖

进入工程目录：

```powershell
cd "D:\python demo\EbeamDataVisualizer"
```

安装依赖：

```powershell
python -m pip install -r requirements.txt
```

生产环境为 Python 3.8 时，请使用 `requirements.txt` 中的版本范围。

## 2. 运行 UI

```powershell
python ui_app.py
```

UI 左侧会根据当前选择的图表类型自动显示需要填写的控件，隐藏无关控件。

基本流程：

1. 点击 `Browse` 选择数据文件。
2. 点击 `Load Columns` 读取字段。
3. 选择 `Chart type`。
4. 设置该图表需要的参数。
5. 点击 `Plot`。
6. 如需保存图片，点击 `Save PNG`。

## 3. 数据字段

推荐字段：

```text
DieIDX
DieIDY
WaferPosX
WaferPosY
ImageID
CD
BendingAngle
LayerType
LayerNO
Moduleindex
```

说明：

| 字段 | 说明 |
|---|---|
| `WaferPosX`, `WaferPosY` | wafer 坐标，300mm wafer 常见范围约 `-150 ~ +150` mm |
| `ImageID` | 拍图 ID |
| `CD` | 默认量测值 |
| `BendingAngle` | 可作为其它 Value column 绘图 |
| `LayerType`, `LayerNO`, `Moduleindex` | 用于区分 line/layer |

旧数据中如果字段叫 `ImageIDX`，程序会兼容读取为 `ImageID`。

## 4. Value Column

`Value column` 是通用量测值字段，不只限于 `CD`。

例如：

- 画 `CD`
- 画 `BendingAngle`
- 画其它数值列

只要字段能转成数字，就可以用于 radius 图、分布图、loading 图和 wafer heatmap。

## 5. Radius 和 Bin

Radius 是虚拟字段，原始数据中不需要有 `Radius` 列。

公式：

```text
Radius = sqrt((WaferPosX - X0)^2 + (WaferPosY - Y0)^2)
```

如果 `WaferPosX/Y` 已经是 `-147 ~ +147` mm，通常：

```text
X0 = 0
Y0 = 0
```

`Radius bin` 是半径分段宽度。300mm wafer、坐标单位为 mm 时建议：

```text
Radius bin = 1, 2, 5, 10
```

不要用 `500`，除非你的坐标单位是 um 级。

## 6. 图表类型

### 6.1 CD by Wafer Radius

用途：看某个数值列随 wafer radius 的变化。

UI 需要设置：

- `Value column`
- `Radius bin`
- `Wafer center`
- 可选 Layer 筛选
- 可选 `Radius chart line only`

输出内容：

- 均值线
- 标准差带
- min/max 点

如果勾选 `Radius chart line only`，只画均值线，图会更干净。

命令：

```powershell
python plot_cd_by_radius.py demo_ebeam_imageid_mm.csv `
  -o cd_radius.png `
  --value-col CD `
  --radius-bin 2 `
  --layer-type M0 `
  --layer-no 1 `
  --line-only
```

### 6.2 Layer CD Distribution

用途：查看不同 layer/line 的分布。

UI 需要设置：

- `Value column`
- `Series display mode`
- `Max series to draw`
- Layer 筛选

命令：

```powershell
python plot_layer_cd_distribution.py demo_ebeam_imageid_mm.csv `
  -o layer_dist.png `
  --value-col CD `
  --layer-type M0 `
  --series-mode overlay
```

### 6.3 Line CD Loading

用途：比较 Line A 和 Line B 的差异，并按 radius 查看 loading。

程序会先在每个 `ImageID + line` 内对多个量测值求均值，再按 `ImageID` 把 A/B 配对。

支持运算：

| Operation | 公式 |
|---|---|
| `subtract` | A - B |
| `add` | A + B |
| `ratio` | A / B |

UI 需要设置：

- `Value column`
- `Radius bin`
- `Line A filters`
- `Line B filters`
- `Operation`

命令：

```powershell
python plot_line_cd_loading.py demo_ebeam_imageid_mm.csv `
  -o loading_radius.png `
  --value-col CD `
  --a-layer-type M0 `
  --a-layer-no 1 `
  --b-layer-type M0 `
  --b-layer-no 2 `
  --operation subtract `
  --radius-bin 2
```

### 6.4 Wafer Value Heatmap

用途：在 wafer map 上查看某个数值列的空间分布。

UI 需要设置：

- `Value column`
- `Wafer diameter`
- `Heatmap bin`
- `Color min`
- `Color max`
- `Map`
- Layer 筛选

300mm wafer：

```text
Wafer diameter = 300
Heatmap bin = 2, 5, 10
```

不同 wafer 对比时，请固定相同的 `Color min` 和 `Color max`。

命令：

```powershell
python plot_wafer_heatmap.py demo_ebeam_imageid_mm.csv `
  -o wafer_cd_heatmap.png `
  --mode value `
  --value-col CD `
  --layer-type M0 `
  --layer-no 1 `
  --bin-size 5 `
  --wafer-diameter 300 `
  --vmin 12 `
  --vmax 15
```

画其它数值列：

```powershell
python plot_wafer_heatmap.py demo_ebeam_imageid_mm.csv `
  -o wafer_bending_heatmap.png `
  --mode value `
  --value-col BendingAngle `
  --layer-type M0 `
  --layer-no 1 `
  --bin-size 5 `
  --wafer-diameter 300
```

### 6.5 Wafer Loading Heatmap

用途：在 wafer map 上查看 Line A/B loading 的空间分布。

UI 需要设置：

- `Value column`
- `Wafer diameter`
- `Heatmap bin`
- `Color min`
- `Color max`
- `Line A filters`
- `Line B filters`
- `Operation`

命令：

```powershell
python plot_wafer_heatmap.py demo_ebeam_imageid_mm.csv `
  -o wafer_loading_heatmap.png `
  --mode loading `
  --value-col CD `
  --a-layer-type M0 `
  --a-layer-no 1 `
  --b-layer-type M0 `
  --b-layer-no 2 `
  --operation subtract `
  --bin-size 5 `
  --wafer-diameter 300 `
  --cmap coolwarm `
  --vmin -0.5 `
  --vmax 0.5
```

### 6.6 Custom X-Y Scatter

用途：自由选择 X/Y 轴画散点图。

UI 需要设置：

- `X axis`
- `Y axis`
- `Sample rows for scatter`
- `Data mode`
- `Scatter mean by ImageID`

建议默认开启 `Scatter mean by ImageID`。这样每个 `ImageID` 每个 series 只画一个平均点，避免一个 ImageID 中多个 CD 值导致点过密。

## 7. Series Display Mode

| 模式 | 说明 |
|---|---|
| `Combine selected data` | 合并所有筛选数据 |
| `Overlay selected series` | 不同 series 叠加在同一坐标系 |
| `Separate panels` | 不同 series 分开画子图 |

## 8. Layer 筛选

全局筛选用于：

- CD by radius
- Layer distribution
- Wafer value heatmap
- Custom scatter

Line A/B 筛选用于：

- Line CD loading
- Wafer loading heatmap

可以点击：

```text
Choose visible values...
Choose Line A values...
Choose Line B values...
```

也可以直接输入逗号分隔值：

```text
M0,M1
1,2
```

## 9. 常见问题

### 9.1 No valid data found

说明当前筛选条件查不到数据。

建议：

- 点击 `Clear visible filters`
- 点击 `Clear Line A/B filters`
- 检查 `LayerType + LayerNO + Moduleindex` 是否是真实存在的组合

### 9.2 CD by radius 图只有一个点

通常是 `Radius bin` 太大。

300mm wafer、坐标单位为 mm 时，建议用：

```text
1, 2, 5, 10
```

### 9.3 不同 wafer heatmap 颜色看起来不能比较

请固定相同的：

```text
Color min
Color max
```

命令行对应：

```text
--vmin
--vmax
```

### 9.4 CSV 被 Excel 打开时报错

关闭 Excel 中打开的 CSV，或复制一份 CSV 再读取。

## 10. 打包 EXE

```powershell
cd "D:\python demo\EbeamDataVisualizer"
.\build_exe.ps1
```

```text
dist\EbeamDataVisualizer\EbeamDataVisualizer.exe
```
