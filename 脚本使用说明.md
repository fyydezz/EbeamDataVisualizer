# Ebeam Data Visualizer 使用说明

本文档面向日常使用者，说明如何安装、运行 UI、选择图表、使用命令行脚本，以及如何处理常见报错。

## 1. 安装依赖

进入工程目录：

```powershell
cd "D:\python demo\EbeamDataVisualizer"
```

安装依赖：

```powershell
python -m pip install -r requirements.txt
```

生产环境为 Python 3.8 时，请使用 `requirements.txt` 中的版本范围。

## 2. 运行 UI

```powershell
python ui_app.py
```

UI 左侧会根据当前选择的图表类型自动显示需要填写的控件，隐藏无关控件。

基本流程：

1. 点击 `Browse` 选择数据文件。
2. 点击 `Load Columns` 读取字段。
3. 选择 `Chart type`。
4. 设置该图表需要的参数。
5. 点击 `Plot`。
6. 如需保存图片，点击 `Save PNG`。

## 3. 数据字段

推荐字段：

```text
DieIDX
DieIDY
WaferPosX
WaferPosY
ImageID
CD
BendingAngle
LayerType
LayerNO
Moduleindex
```

说明：

| 字段 | 说明 |
|---|---|
| `WaferPosX`, `WaferPosY` | wafer 坐标，300mm wafer 常见范围约 `-150 ~ +150` mm |
| `ImageID` | 拍图 ID |
| `CD` | 默认量测值 |
| `BendingAngle` | 可作为其它 Value column 绘图 |
| `LayerType`, `LayerNO`, `Moduleindex` | 用于区分 line/layer |

旧数据中如果字段叫 `ImageIDX`，程序会兼容读取为 `ImageID`。

## 4. Value Column

`Value column` 是通用量测值字段，不只限于 `CD`。

例如：

- 画 `CD`
- 画 `BendingAngle`
- 画其它数值列

只要字段能转成数字，就可以用于 radius 图、分布图、loading 图和 wafer heatmap。

## 5. Radius 和 Bin

Radius 是虚拟字段，原始数据中不需要有 `Radius` 列。

公式：

```text
Radius = sqrt((WaferPosX - X0)^2 + (WaferPosY - Y0)^2)
```

如果 `WaferPosX/Y` 已经是 `-147 ~ +147` mm，通常：

```text
X0 = 0
Y0 = 0
```

`Radius bin` 是半径分段宽度。300mm wafer、坐标单位为 mm 时建议：

```text
Radius bin = 1, 2, 5, 10
```

不要用 `500`，除非你的坐标单位是 um 级。

## 6. 图表类型

### 6.1 CD by Wafer Radius

用途：看某个数值列随 wafer radius 的变化。

UI 需要设置：

- `Value column`
- `Radius bin`
- `Wafer center`
- 可选 Layer 筛选
- 可选 `Radius chart line only`

输出内容：

- 均值线
- 标准差带
- min/max 点

如果勾选 `Radius chart line only`，只画均值线，图会更干净。

命令：

```powershell
python plot_cd_by_radius.py demo_ebeam_imageid_mm.csv `
  -o cd_radius.png `
  --value-col CD `
  --radius-bin 2 `
  --layer-type M0 `
  --layer-no 1 `
  --line-only
```

### 6.2 Layer CD Distribution

用途：查看不同 layer/line 的分布。

UI 需要设置：

- `Value column`
- `Series display mode`
- `Max series to draw`
- Layer 筛选
- `Series colors`（可选，用于指定每一根分布线的颜色）

图例说明：

- 每一根线的 label 会自动显示 `μ`、`σ` 和 `n`
- `μ` 是该 series 的均值
- `σ` 是该 series 的样本标准差
- `n` 是该 series 参与分布统计的数据量

自定义每根线颜色：

```text
# 按图例顺序指定
#d62728,#1f77b4,#2ca02c

# 按 label 片段指定，适合 wafer-to-wafer 或 layer-to-layer 对比
LayerNO=1:#d62728;LayerNO=2:#1f77b4;LayerNO=3:#2ca02c
```

说明：

- `Series colors` 修改后会自动刷新当前图，不需要重新点击 `Plot`
- 使用 `label片段:颜色` 时，只要片段能匹配图例文字即可，例如 `LayerNO=1`
- 颜色支持 Matplotlib 常见格式，例如 `#d62728`、`red`、`tab:blue`

命令：

```powershell
python plot_layer_cd_distribution.py demo_ebeam_imageid_mm.csv `
  -o layer_dist.png `
  --value-col CD `
  --layer-type M0 `
  --series-mode overlay `
  --series-colors "LayerNO=1:#d62728;LayerNO=2:#1f77b4;LayerNO=3:#2ca02c"
```

### 6.3 Radius Sigma by Wafer Radius

用途：查看某个量测值随 wafer radius 的离散程度变化，Y 轴为 `σ`。

UI 需要设置：

- `Value column`
- `Radius bin`
- `Sigma settings`
- `Series display mode`
- Layer 筛选
- `Series colors`（可选）

`Sigma settings` 支持：

| Mode | 说明 |
|---|---|
| `radius_bin` | 直接在每个 radius bin 内计算所有点的样本标准差 |
| `image` | 先在每个 `ImageID + series` 内计算 σ，再按 radius bin 聚合 image-level σ |

命令：

```powershell
python plot_radius_sigma.py demo_ebeam_imageid_mm.csv `
  -o radius_sigma.png `
  --value-col CD `
  --radius-bin 2 `
  --sigma-mode radius_bin `
  --layer-type M0 `
  --series-mode overlay
```

### 6.4 Line CD Loading

用途：比较 Line A 和 Line B 的差异，并按 radius 查看 loading。

程序会先在每个 `ImageID + line` 内对多个量测值求均值，再按 `ImageID` 把 A/B 配对。

支持运算：

| Operation | 公式 |
|---|---|
| `subtract` | A - B |
| `add` | A + B |
| `ratio` | A / B |

UI 需要设置：

- `Value column`
- `Radius bin`
- `Line A filters`
- `Line B filters`
- `Operation`

命令：

```powershell
python plot_line_cd_loading.py demo_ebeam_imageid_mm.csv `
  -o loading_radius.png `
  --value-col CD `
  --a-layer-type M0 `
  --a-layer-no 1 `
  --b-layer-type M0 `
  --b-layer-no 2 `
  --operation subtract `
  --radius-bin 2
```

### 6.5 FIN Center Loading

用途：针对上中下三根 FIN 的场景，计算中间 FIN 相对于上下 FIN 平均值的 loading。

公式：

```text
((Line A + Line C) / 2) - Line B
```

典型设置：

- Line A：上 FIN，例如 `LayerNO=1`
- Line B：中 FIN，例如 `LayerNO=2`
- Line C：下 FIN，例如 `LayerNO=3`

程序会先在每个 `ImageID + line` 内对多条量测值求均值，再按 `ImageID` 对齐 A/B/C 后计算公式。

命令：

```powershell
python plot_fin_center_loading.py demo_ebeam_imageid_mm.csv `
  -o fin_center_loading.png `
  --value-col CD `
  --a-layer-type FIN `
  --a-layer-no 1 `
  --b-layer-type FIN `
  --b-layer-no 2 `
  --c-layer-type FIN `
  --c-layer-no 3 `
  --radius-bin 2 `
  --series-mode overlay
```

### 6.6 Wafer Value Heatmap

用途：在 wafer map 上查看某个数值列的空间分布。

UI 需要设置：

- `Value column`
- `Wafer diameter`
- `Heatmap bin`
- `Color min`
- `Color max`
- `Map`
- Layer 筛选

300mm wafer：

```text
Wafer diameter = 300
Heatmap bin = 2, 5, 10
```

不同 wafer 对比时，请固定相同的 `Color min` 和 `Color max`。

命令：

```powershell
python plot_wafer_heatmap.py demo_ebeam_imageid_mm.csv `
  -o wafer_cd_heatmap.png `
  --mode value `
  --value-col CD `
  --layer-type M0 `
  --layer-no 1 `
  --bin-size 5 `
  --wafer-diameter 300 `
  --vmin 12 `
  --vmax 15
```

画其它数值列：

```powershell
python plot_wafer_heatmap.py demo_ebeam_imageid_mm.csv `
  -o wafer_bending_heatmap.png `
  --mode value `
  --value-col BendingAngle `
  --layer-type M0 `
  --layer-no 1 `
  --bin-size 5 `
  --wafer-diameter 300
```

### 6.7 Wafer Loading Heatmap

用途：在 wafer map 上查看 Line A/B loading 的空间分布。

UI 需要设置：

- `Value column`
- `Wafer diameter`
- `Heatmap bin`
- `Color min`
- `Color max`
- `Line A filters`
- `Line B filters`
- `Operation`

命令：

```powershell
python plot_wafer_heatmap.py demo_ebeam_imageid_mm.csv `
  -o wafer_loading_heatmap.png `
  --mode loading `
  --value-col CD `
  --a-layer-type M0 `
  --a-layer-no 1 `
  --b-layer-type M0 `
  --b-layer-no 2 `
  --operation subtract `
  --bin-size 5 `
  --wafer-diameter 300 `
  --cmap coolwarm `
  --vmin -0.5 `
  --vmax 0.5
```

### 6.8 Custom X-Y Scatter

用途：自由选择 X/Y 轴画散点图。

UI 需要设置：

- `X axis`
- `Y axis`
- `Sample rows for scatter`
- `Data mode`
- `Scatter mean by ImageID`

建议默认开启 `Scatter mean by ImageID`。这样每个 `ImageID` 每个 series 只画一个平均点，避免一个 ImageID 中多个 CD 值导致点过密。

## 7. Series Display Mode

| 模式 | 说明 |
|---|---|
| `Combine selected data` | 合并所有筛选数据 |
| `Overlay selected series` | 不同 series 叠加在同一坐标系 |
| `Separate panels` | 不同 series 分开画子图 |

## 8. Plot Style 和 Series Colors

`Plot style` 用于调整已经画出的图。多数样式项修改后会自动刷新，不需要重新查询数据。

常用项：

| 项目 | 说明 |
|---|---|
| `Line color` | 第一根线的默认颜色 |
| `Series colors` | 多根线分别指定颜色 |
| `Line width` | 线宽 |
| `Scatter color` | 散点图第一组点的默认颜色 |
| `Scatter size` | 散点大小 |
| `Scatter alpha` | 散点透明度，0 到 1 |
| `X label`, `Y label` | 自定义坐标轴名称 |
| `Axis label size` | 坐标轴文字大小 |
| `Axis label weight` | 坐标轴文字粗细 |

`Series colors` 支持两种写法：

```text
# 按当前图例顺序依次套用
#d62728,#1f77b4,#2ca02c

# 按图例 label 片段指定
LayerNO=1:#d62728;LayerNO=2:#1f77b4;LayerNO=3:#2ca02c
```

做 wafer-to-wafer 对比时，建议为相同 wafer 或相同 layer 固定一套颜色，避免不同图片之间颜色含义变化。

## 9. Layer 筛选

全局筛选用于：

- CD by radius
- Layer distribution
- Wafer value heatmap
- Custom scatter

Line A/B 筛选用于：

- Line CD loading
- Wafer loading heatmap

Line A/B/C 筛选用于：

- FIN center loading

可以点击：

```text
Choose visible values...
Choose Line A values...
Choose Line B values...
```

也可以直接输入逗号分隔值：

```text
M0,M1
1,2
```

## 10. 常见问题

### 10.1 No valid data found

说明当前筛选条件查不到数据。

建议：

- 点击 `Clear visible filters`
- 点击 `Clear Line A/B filters`
- 检查 `LayerType + LayerNO + Moduleindex` 是否是真实存在的组合

### 10.2 CD by radius 图只有一个点

通常是 `Radius bin` 太大。

300mm wafer、坐标单位为 mm 时，建议用：

```text
1, 2, 5, 10
```

### 10.3 不同 wafer heatmap 颜色看起来不能比较

请固定相同的：

```text
Color min
Color max
```

命令行对应：

```text
--vmin
--vmax
```

### 10.4 CSV 被 Excel 打开时报错

关闭 Excel 中打开的 CSV，或复制一份 CSV 再读取。
